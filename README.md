## RMI

Para rodar o exemplo em questão, é preciso que o serviço de nomes do RMI esteja em execução.

** Iniciando o rmimregistry **
- Abra o Prompt de Comando e va ate a pasta bin onde estao os arquivos .class
- Inicie o rmiregistry da seguinte forma: ```rmiregistry -J-Djava.rmi.server.useCodebaseOnly=false```

** Servidor RMI **
- Execute a classe NotepadRMIServer

** Teste com cliente RMI **
- Execute a classe MainRMIClientExample
